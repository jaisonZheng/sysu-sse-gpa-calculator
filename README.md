## 📖 项目故事

有一天，一位中山大学软件工程学院的同学（也就是本项目的甲方爸爸）在深夜面对教务系统陷入了沉思：

> "这保研绩点到底怎么算啊？！😫"

教务系统显示的是普通GPA，但保研用的是**综合成绩绩点**，公式长这样：

```
综合成绩绩点 = (所有公共必修课和一年级专业必修课的平均绩点 × 0.5)
             + (二年级和三年级专业必修课的平均绩点 × 0.5)
```

手动算？不存在的，课程那么多，还要按学年分类，还要学分加权平均...

于是，这个网站诞生了！✨

## 🚀 功能特性

### 🔐 自动模式（懒人福音）
- 输入你的 NetID 和密码
- 系统自动登录教务系统
- 一键爬取所有学年成绩
- 自动分类计算保研绩点
- **密码不会被存储**，仅用于登录

### ✏️ 手动模式（掌控一切）
- 选择你的入学年份（2022/2023/2024/2025级）
- 系统自动加载对应培养方案
- 输入成绩即可计算
- 支持添加/删除/修改课程

### 🎯 智能特性
- **学年自动判断**：根据当前日期自动显示可用学年
- **培养方案适配**：不同年级有不同的培养方案
- **实时绩点显示**：输入成绩立即显示对应绩点

## 📝 计算公式详解

### 第一步：分类
- **公必**：公共必修课（所有年级都算）
- **专必**：专业必修课
- **专选**：专业选修课（不算入保研绩点）
- **公选**：公共选修课（不算入保研绩点）

### 第二步：加权平均
```
Part 1 = (所有公必绩点 × 学分 + 大一专必绩点 × 学分) / 总学分
Part 2 = (大二专必绩点 × 学分 + 大三专必绩点 × 学分) / 总学分

保研绩点 = Part 1 × 0.5 + Part 2 × 0.5
```
